Register a goal callback to fire at a lifecycle or channel event. Step phrasings starting with "on", "before", or "after" (e.g. "on ask on input channel call X", "before step call X", "after write call X") map here, NOT to output.ask, channel.set, or environment.run. The `Trigger` parameter is a `trigger` enum member emitted as a bare string (see Type Information for the members). Channel events (the read/write/ask family) also take a `ChannelName` filter.
