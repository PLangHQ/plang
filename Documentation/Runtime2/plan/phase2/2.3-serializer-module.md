# Phase 2.3 -- Serializer Module (`serializer/*`)

**Status:** NOT STARTED
**Complexity:** Medium
**Depends on:** Phase 1

## Goal
Provide format conversion between JSON, XML, CSV, YAML etc.

## Design
- Handlers: `toJson`, `fromJson`, `toXml`, `fromXml`, `toCsv`, `fromCsv`, `toYaml`, `fromYaml`
- **Leverages:** Existing SerializerRegistry infrastructure
- **Pluggable converters:** PLang programmer can register custom converters via goals
- **Key role:** Enables the type-aware conversion pipeline from Phase 0 (Data.Type drives which serializer is used)

## Files
- New: `PLang/Runtime2/modules/serializer/*.cs`

## Tests
- C# + PLang integration for each format

## Verification
- Round-trip: serialize then deserialize preserves data
- Data.Type drives automatic format selection
- Custom converter registration works
