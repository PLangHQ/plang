# Phase 1.1 -- Loop Module (`loop/foreach`)

**Status:** NOT STARTED
**Complexity:** Medium
**Depends on:** Phase 0

## Goal
Iterate collections by calling a goal per item.

## Design
- **Handler:** `loop/foreach.cs` - iterate collection, call goal per item
- **Parameters:** collection (list/dict), goalToCall (GoalCall), itemVariableName (string)
- **PLang syntax:** `foreach %items%, call ProcessItem item=%item%`
- foreach always calls a goal (no sub-steps - this is intentional, keep it)
- **Collection handling:**
  - empty/null collection = count 0, just skip (no error)
  - single object (not a list) = treated as list with count 1
- **Future:** multithreaded foreach (discuss when we get there, not now)

## Files
- New: `PLang/Runtime2/modules/loop/foreach.cs`

## Tests
- C# unit tests: valid/invalid inputs, type preservation
- PLang integration: `Tests/Runtime2/Loop/`

## Verification
- foreach over list calls goal per item
- empty/null collection skips without error
- single object treated as list of 1
