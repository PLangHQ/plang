# Phase 1.5 -- Math Module (`math/*`)

**Status:** NOT STARTED
**Complexity:** Medium (NCalc does the heavy lifting)
**Depends on:** Phase 0

## Goal
Provide math operations using the NCalc expression engine.

## Design
- **Use NCalc library** - battle-tested, supports complex formula strings
- Handlers: `calculate` (NCalc expression), plus convenience: `round`, `floor`, `ceiling`, `min`, `max`, `random`, `abs`
- NCalc allows users to write formulas like `- calculate '(%price% * %quantity%) * (1 + %taxRate%)', write to %total%`

## Files
- New: `PLang/Runtime2/modules/math/*.cs`

## Tests
- C# unit tests: each handler with various inputs
- PLang integration: `Tests/Runtime2/Math/`

## Verification
- NCalc expressions evaluate correctly with variable substitution
- Convenience handlers (round, floor, etc.) work with edge cases
