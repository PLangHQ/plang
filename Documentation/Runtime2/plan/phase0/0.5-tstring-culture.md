# Phase 0.5 — TString Skeleton & CultureInfo

## Goal
Create the TString (translatable string) type and register it in TypeMapping. Add CultureInfo to PLangAppContext. This is design-only — no actual translation lookup yet. Establishes the foundation so future modules can build on it.

## Depends On
- Nothing — independent of other Phase 0 sub-phases

## Changes

### 1. Create `PLang/Runtime2/Memory/TString.cs`
```csharp
using System.Globalization;

namespace PLang.Runtime2.Memory;

/// <summary>
/// Translatable string. Wraps a string value with a translation key.
/// When translation is configured, ToString() returns the localized version.
/// For now, returns the raw Value (translation lookup is Phase 6+).
/// </summary>
public sealed class TString
{
    /// <summary>The original string value (used as fallback and default language).</summary>
    public string Value { get; }

    /// <summary>Translation key for lookup (e.g., hash or developer-defined key).</summary>
    public string? Key { get; init; }

    /// <summary>Optional named parameters for string interpolation after translation.</summary>
    public Dictionary<string, object?>? Parameters { get; init; }

    public TString(string value, string? key = null)
    {
        Value = value;
        Key = key;
    }

    // TODO: When translation service exists, this will do culture-aware lookup.
    // For now, just return the raw value.
    public override string ToString() => Value;

    // Implicit conversions for ease of use
    public static implicit operator TString(string value) => new(value);
    public static implicit operator string(TString ts) => ts.ToString();

    public override bool Equals(object? obj) => obj switch
    {
        TString other => string.Equals(Value, other.Value, StringComparison.Ordinal),
        string str => string.Equals(Value, str, StringComparison.Ordinal),
        _ => false
    };

    public override int GetHashCode() => Value.GetHashCode();
}
```

### 2. Register in `PLang/Runtime2/Utility/TypeMapping.cs`

Add to the `NameToType` dictionary:
```csharp
{ "tstring", typeof(TString) },
{ "translatable", typeof(TString) },
```

Add to the `TypeToName` dictionary:
```csharp

```

### 3. Add CultureInfo to `PLang/Runtime2/Context/PLangAppContext.cs`

Add property (needs `using System.Globalization;`):
```csharp
/// <summary>
/// Application culture for formatting dates, numbers, etc.
/// Defaults to InvariantCulture. Set via PLang settings or startup.
/// </summary>
public CultureInfo Culture { get; set; } = CultureInfo.InvariantCulture;
```
