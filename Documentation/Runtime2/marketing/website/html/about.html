From English to running software

AI is the compiler, not a dependency. It reads your intent once, then gets out of the way.

1

Write

.goal files

Write plain English steps describing what your app should do. These are your source code — version them, diff them, read them.

2

Build

plang build

An LLM reads each step once and compiles it to a .pr file — inspectable JSON. Only changed steps rebuild.

AI runs here
3

Run

plang

Your app executes the compiled instructions directly. Zero AI calls. Zero network. Runs offline, deterministically.
