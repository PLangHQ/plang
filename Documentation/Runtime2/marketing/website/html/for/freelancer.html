You're a one-person shop juggling five clients, three deadlines, and an invoicing system that's really just a Google Sheet you copy every month. Time tracking lives in your head. Sending invoices means opening a template, changing the date, exporting to PDF, emailing it — every single time.

The PLang way

Start.goal
Start- start webserver
LogTime - POST- insert into timeEntries, clientId=%request.clientId%,    hours=%request.hours%, description=%request.description%,    date=%Now%, billed='no', write to %id%- write out {entryId: %id%, status: 'logged'}
MonthlyInvoicing- every 1st of month at 9am, call !GenerateInvoices
GenerateInvoices- select distinct clientId from timeEntries where billed='no',    write to %clients%- foreach %clients%, call !GenerateInvoice item=%client%
GenerateInvoice- select * from clients where id=%client.clientId%, write to %clientInfo%- select sum(hours) as totalHours from timeEntries    where clientId=%client.clientId% and billed='no', write to %summary%- set %total% = %summary.totalHours% * %clientInfo.rate%- send email to %clientInfo.email%, subject "Invoice for %clientInfo.name% - %Now%",    body "Hours: %summary.totalHours%\nRate: $%clientInfo.rate%/hr\nTotal: $%total%\n\nThank you for your business."- update timeEntries set billed='yes' where clientId=%client.clientId% and billed='no'
AddClient - POST- insert into clients, name=%request.name%, email=%request.email%,    rate=%request.rate%, write to %id%- write out {clientId: %id%, status: 'added'}
Dashboard - GET- select clientId, sum(hours) as unbilledHours from timeEntries    where billed='no' group by clientId, write to %unbilled%- select count(*) as sent, sum(total) as revenue from invoices    where created >= %Now-30days%, write to %invoiceStats%- write out {unbilledHours: %unbilled%, invoices: %invoiceStats%}

That's your freelance business system.

Time tracking, automatic monthly invoicing, client management, dashboard. Invoices calculated and sent automatically on the 1st.

What just happened

  • insert into timeEntriesTime tracking database created automatically. Log hours from any device.
  • every 1st of monthMonthly invoicing runs automatically. No reminders needed.
  • sum(hours) * rateInvoice totals calculated from logged time. No manual spreadsheet math.
  • send emailProfessional invoices emailed automatically to each client.
  • Dashboard - GETUnbilled hours and invoice stats at a glance from any browser.

Need to change your rate for a client? Update one row. Need weekly invoicing instead? Change one line.

The build / run split

PLang uses AI at build time to compile your English into executable instructions. At runtime, no AI — your freelance system runs reliably with zero ongoing cost. Build once, your invoicing and tracking run forever.
