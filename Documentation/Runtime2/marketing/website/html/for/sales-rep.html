Automate follow-ups.
Focus on closing.

Your CRM has 200 contacts that need follow-ups. You spend the first hour of every day scrolling through leads, copy-pasting email templates, logging activities. By the time you're done with admin, half your selling time is gone. That prospect who was "very interested" last Tuesday? You forgot to call them back.

The PLang way

Start.goal
Start- every day at 7am, call !MorningFollowUps- every friday at 5pm, call !WeeklyPipeline
MorningFollowUps- select * from leads where nextFollowUp <= %Now%    and status='active', write to %dueLeads%- foreach %dueLeads%, call !FollowUp item=%lead%
FollowUp- send email to %lead.email%,    subject "Checking in, %lead.name%",    body "Hi %lead.name%, wanted to follow up on %lead.interest%. Any questions?"- update leads set lastContact=%Now%,    nextFollowUp=%Now+3days%,    touchCount=%lead.touchCount%+1 where id=%lead.id%- insert into activity_log, leadId=%lead.id%,    type='email', date=%Now%, notes='Auto follow-up sent'
WeeklyPipeline- select status, count(*) as count, sum(dealValue) as value    from leads group by status, write to %pipeline%- select name, dealValue, status from leads    where status='proposal' order by dealValue desc, write to %hotDeals%- send email to me@company.com,    subject "Weekly Pipeline - %Now%",    body "Pipeline:\n%pipeline%\n\nHot Deals:\n%hotDeals%"
AddLead - POST- insert into leads, name=%request.name%, email=%request.email%,    interest=%request.interest%, dealValue=%request.value%,    status='new', nextFollowUp=%Now+1day%,    lastContact=%Now%, touchCount=0- write out {status: 'added'}

That's your personal sales assistant.

Automatic follow-ups on schedule, activity logging, weekly pipeline reports, and new lead intake. All in plain English you can adjust yourself.

What just happened

  • every day at 7amFollow-ups run before you pour your coffee.
  • select where nextFollowUp <= %Now%Finds every lead due for contact today.
  • send emailPersonalized follow-ups sent directly. No template tool.
  • update set nextFollowUp=%Now+3days%Automatically schedules the next touch.
  • insert into activity_logEvery interaction logged automatically.
  • WeeklyPipelinePipeline report delivered every Friday.

The build / run split

PLang uses AI at build time to compile your English into executable instructions. At runtime, no AI — your follow-up system runs reliably every morning with no ongoing cost.

Get started

# Download from GitHub releases, unzip, add to PATH

# Create Start.goal with your sales workflow
