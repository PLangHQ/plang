Your practice runs on deadlines — filing dates, response windows, statute of limitations. Missing one isn't embarrassing, it's malpractice. So you track them manually: calendar entries, sticky notes, a spreadsheet. Document assembly is another time sink — template, find-and-replace client name, case number, dates, email for review. You bill $300/hour and you're spending it on copy-paste.

The PLang way

Start.goal
Start- start webserver- every day at 7am, call !CheckDeadlines
NewCase - POST- insert into cases, clientName=%request.clientName%,    caseNumber=%request.caseNumber%, type=%request.type%,    filedDate=%Now%, status='active', write to %caseId%- call !SetDeadlines caseId=%caseId%, filedDate=%Now%- write out {caseId: %caseId%, status: 'created'}
SetDeadlines- insert into deadlines, caseId=%caseId%, type='response',    dueDate=%filedDate+30days%, status='pending'- insert into deadlines, caseId=%caseId%, type='filing',    dueDate=%filedDate+60days%, status='pending'- insert into deadlines, caseId=%caseId%, type='review',    dueDate=%filedDate+90days%, status='pending'
CheckDeadlines- select deadlines.*, cases.clientName, cases.caseNumber    from deadlines join cases on deadlines.caseId = cases.id    where dueDate <= %Now+7days% and status='pending',    write to %upcoming%- foreach %upcoming%, call !AlertDeadline item=%deadline%
AlertDeadline- send email to attorney@firm.com,    subject "Deadline: %deadline.type% - Case #%deadline.caseNumber%",    body "%deadline.type% deadline for %deadline.clientName% (Case #%deadline.caseNumber%) is due on %deadline.dueDate%."
GenerateDocument - POST- select * from cases where id=%request.caseId%, write to %case%- read file templates/%request.templateName%.txt, write to %template%- set %document% = %template%    replace "{CLIENT_NAME}" with %case.clientName%,    replace "{CASE_NUMBER}" with %case.caseNumber%,    replace "{DATE}" with %Now%- save %document% to documents/%case.caseNumber%_%request.templateName%.txt- send email to %case.clientEmail%,    subject "Document for Case #%case.caseNumber%",    body "Please review the attached document for your case."

That's your case management system.

Case intake, automatic deadline tracking, daily alerts, document generation with client delivery. All in plain English.

What just happened

  • insert into cases/deadlinesCase database and deadline tracker created automatically. No software to configure.
  • every day at 7amDeadline checks run before you reach the office. Never miss a filing date.
  • send emailDeadline reminders delivered to your inbox. Professional and on time.
  • read templates/...Document templates are text files you control. No locked-down software.
  • NewCase - POSTAn API endpoint. Connect intake forms and stop manual data entry.

Need to change the response window from 30 to 21 days? Change one line. Need a new document template? Add a text file.

The build / run split

PLang uses AI at build time to compile your English into executable instructions. At runtime, no AI — your case management system runs reliably with zero ongoing cost. Build once, your legal workflows run forever.
