# Phase 5.4 -- PLang-Written Modules (in /system/)

**Status:** NOT STARTED
**Complexity:** Medium
**Depends on:** Phase 3

## Goal
Identify and implement modules that can be written as PLang goals instead of C#.

## Design
Consider which modules can be written as PLang goals instead of C#:
- Logger (could be event binding + output.write)
- Assert (could be condition + error/throw)
- Validate (could be condition + error/throw)
- Others where the underlying primitives exist

## Files
- New: `/system/modules/<module>/*.goal` for each PLang-written module

## Tests
- PLang integration tests

## Verification
- PLang-written modules produce identical behavior to C# equivalents
- Performance is acceptable
