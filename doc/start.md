# PLang Runtime — Developer Reference

Documentation mirrors the OBP object tree. Every folder is a concept; `start.md` is its entry point.

## Top-level concepts

- [app/goal/](app/goal/start.md) — Goals: the named entry points of a PLang program
- [app/goal/step/](app/goal/step/start.md) — Steps: the individual instructions inside a goal
- [app/type/](app/type/start.md) — Values: the type system, kind, binary lazy decode, list/dict slots
