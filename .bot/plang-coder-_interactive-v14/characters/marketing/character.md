# PLang — Programming in Plain Language

**What is PLang?** A programming language where you write in natural language — no syntax to memorize, no brackets to match, no semicolons to hunt down. Just describe what you want, and PLang makes it happen.

**Who is it for?** Anyone with an idea. Whether you're a founder prototyping a product, a domain expert automating workflows, or a developer tired of boilerplate — PLang lets you build real software by writing steps in plain English.

**How it works:**
- Write your logic as simple goal files — step-by-step instructions in natural language
- PLang compiles your intent into executable code using an LLM at build time
- Your app runs locally, privately, with no AI dependency at runtime
- Built-in modules handle databases, HTTP, files, crypto, email, and more — no packages to install

**What makes PLang different:**
- **Natural language is the source code** — not a wrapper around code, not a chatbot. Your `.goal` files ARE the program
- **Build once, run without AI** — the LLM is only used at compile time. Your app runs independently
- **Identity built in** — every PLang app has cryptographic identity baked in. No auth libraries, no token management
- **SQLite by default** — data storage works out of the box with zero configuration
- **Deterministic execution** — same input, same output. Natural language in, reliable software out

**The pitch in one line:** Stop writing code. Start writing what you mean.
