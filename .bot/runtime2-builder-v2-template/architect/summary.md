# Builder V2 — Piece 6: UI Module (Template Rendering)

**v1**: UI module with Liquid rendering via Fluid. Single `render` action (detects file vs inline). Memory stack accessible by default in templates. Custom tags: `callGoal`, partials via Fluid built-in `include`. No separate template module — rendering is a UI capability. DOM/layout/events deferred to future piece. See [v1/plan.md](v1/plan.md).
